Web application generator count of monthly hours in *. Xls


DEMO:	http://genparte.disastercode.com.es


Repository directories:
	web: contains the web application.
	ws: contains a web service through which you can get holidays.


Technologies used:
	- Jquery Plugin Fullcalendar to implement the calendar.
			http://arshaw.com/fullcalendar/


	- Nusoap, as web server for holidays.
			http://sourceforge.net/projects/nusoap/


	- PHPExcel, library for generating excel sheet.
			http://phpexcel.codeplex.com/


	- Forms Design: Formly -> 
				http://thrivingkings.com/read/Formly-The-form-glamorizer-for-jQuery
			      NiceForms -> 
				http://www.emblematiq.com/lab/niceforms/
